About

Fledge aims to automate hashtag generation for any potential twitter handle, eliminating the need to contemplate about inputting tags every time they Tweet. It aims to classify raw text data into five categories.

Product Features
  • Generate hashtags serving general purpose

    Assists users to use a set of hashtags by processing their images.

  • Text categorization

    Classifies text into five main categories : sports, business, education, entertainment and technology.

  • Generate hashtags serving commercial purpose

    Solution for social media business users to generate trending and relevant hashtags.
